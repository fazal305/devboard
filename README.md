# DevBoard

A dark neon Trello-lite project management board built with HTML, CSS, Bootstrap 5, and vanilla JavaScript.

DevBoard lets users create multiple boards, add custom columns, create task cards, drag cards between columns, search cards, track stats, clear completed work, and save everything automatically in localStorage.

## Features

- Multiple project boards
- Default kanban columns
- Custom columns
- Create, edit, and delete cards
- Priority badges: LOW, MEDIUM, HIGH
- Due dates
- Neon color labels
- Drag and drop between columns
- Search cards by title
- Stats bar with total and column counts
- Clear DONE column
- Light/dark theme toggle
- localStorage persistence
- Responsive layout

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- Vanilla JavaScript
- localStorage
- HTML5 Drag and Drop API
