# DevBoard

A dark, responsive Kanban-style project board built with HTML, CSS, and vanilla JavaScript.

DevBoard is a Trello-lite productivity app for managing project cards across multiple boards. It supports custom boards, columns, task cards, priorities, due dates, drag-and-drop movement, search, stats, theme switching, and automatic browser saving with `localStorage`.

## Live Demo

https://fazal305.github.io/devboard/

## Features

- Multiple project boards
- Default Kanban columns
- Custom columns
- Create, edit, and delete task cards
- Priority badges for low, medium, and high tasks
- Due date support
- Color labels for cards
- Drag and drop cards between columns
- Search cards by title
- Stats bar with total and per-column card counts
- Clear completed work from the Done column
- Dark and light theme toggle
- Automatic `localStorage` persistence
- Responsive desktop and mobile layout
- Safer DOM rendering for user-created text

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API
- HTML5 Drag and Drop API
